Frontend Developer

El Frontend se refiere a la parte del desarrollo web que esta orientado al diseño de un sitio web. Algunas herramientas con las que trabaje en proyectos frontend HTML, CSS, Sass, Bootstrap, JavaScript.

Backend Developer

El Backend se refiere a la capa de programacion ejecutada en el navegador. Es la logica de la tecnología que hace que una pagina web funcione.
